Discover Food Bundles¶

A food confectionary is undergoing a tough time in sales and wants use machine learning to turn their fortunes around. They have captured their daily transaction data from 30th Oct 2016 to 9th Apr 2017 and want to understand what are all the food bundles that are popular among their customers.

Your job is to perform Market Basket Analysis using the Apriori algorithm and recommend the confectionary the food bundles that they should be marketing more.

Below is the list of questions you need to crack as part of this assignment

  1. Read the data from the csv file into a dataframe.

  2. Convert the Datetime object column to a datetime column and set it as index.

  3. Identify the top-10 best-selling and worst-selling items in the confectionary.

  4. Find the day and the timeslot in which most of the sales are happening in the confectionary. Hint:You need to extract the hour and day details from the Datetime column.

  5. Convert the data into a one-hot encoded format containing the count of items sold in a transaction. Hint: You need to create the Count column to track the number of items sold in a transaction. You will then use the pivot_table method to create the one-hot encoded format.

  6. Encode the data into 1’s and 0’s to apply the apriori algorithm. Hint:Result obtained in the above step will contain the actual number of items sold in a transaction. In order to apply Apriori, we just need to know whether an item is present in a transaction or not.

  7. Generate frequent itemsets and association rules with a minimum support threshold of 1% and a minimum lift value of 1. Sort them by confidence values.

  8. What percentage of transactions involved customers buying ‘Sandwich’ and ‘Coffee’? Hint: Select only those association rules that have lift value greater than 1 and confidence value greater than 0.5 to generate the insights.

  9. What is the probability that a customer ‘Coffee’ given that he bought a ‘Juice’?

  10. Which item lifts the Coffee’s purchase by the most?

#Create a 'Count' column to get the number of items sold per transaction 
confec = confec.groupby(["Transaction","Item"]).size().reset_index(name="Count")
confec.head()

#Pivot the data into a one-hot encoded format
confec_basket = pd.pivot_table(confec, index='Transaction', columns='Item',values='Count', fill_value=0)
confec_basket.shape

confec_basket.head()

Note: At this stage, the one-hot encoded table shows the count of items purchased as result. We just need to know whether an item is present in a transaction or not.

Step 6: Encode the data into 1’s and 0’s to apply the apriori algorithm.¶

# the encoding function
def encode_units(x):
    if x <= 0:
        return 0
    if x >= 1:
        return 1

confec_basket_sets = confec_basket.applymap(encode_units)

confec_basket_sets.head()

After applying the encoding function, the cell value for all the items has become either a “1” or a “0”, which is what we need for the Apriori function.

Step 7:Generate frequent itemsets and association rules with a minimum support threshold of 1% and a minimum lift value of 1. Sort them by confidence values.¶

frequent_itemsets = apriori(confec_basket_sets, min_support=0.01, use_colnames=True)

#generate association rules and sort them by confidence in ascending order
rules = association_rules(frequent_itemsets, metric="lift", min_threshold=1)
rules.sort_values("confidence", ascending = False, inplace = True)
rules.head(10)

Step 8: What percentage of transactions involved customers buying ‘Sandwich’ and ‘Coffee’?¶

#select only those rules which have lift > 1 and confidence > 0.5
rules[ (rules['lift'] >= 1) &
       (rules['confidence'] >= 0.5) ]

From the above table, you can observe that the support value for the association rule involving Sandwich and Coffee is 0.038. This means out of all the transactions, 3.8% of the transactions involved customers buying Sandwich and Coffee

Step 9: What is the probability that a customer ‘Coffee’ given that he bought a ‘Juice’?¶

From the above association rule table, you can observe that the confidence value for the association rule involving Juice and Coffee is 0.532. This implies that whenever a customer buys Juice, there is a 53.2% chance that he will also buy ‘Coffee’.

Step 10:Which item lifts the Coffee’s purchase by the most?¶

If you observe all the association rules in the above table, you can notice that the lift value(1.47) is the highest for the rule having ‘Toast’ as the antecedent and ‘Coffee’ as the consequent. This implies that the purchase of ’Toast’ lifts the chances of someone buying ‘Coffee’ by 1.47 times.
